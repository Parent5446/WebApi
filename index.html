WebApi by Parent5446

A set of abstraction classes to allow programmers to create websites with extreme ease.

This is a new type of PHP API. While a lot of web designers are looking for an API that makes their website a little easier to draft up, this takes over the entire process. For websites that do not require any custom programming or specific classes, this API can create almost any website with only one file of customized code.

Dependencies

php5 mysql-server

Install

1. Unpack the source into the directory of your choice. 2. Edit the configuration file. 3. Type custom controllers in the custom.php file.
